### Introduction

This Repo contains the Robotdog Project (As Featured at California State Universitiy Fullerton's 2026 ECS Innovations Expo) 

### Table of Contents

1. DJI Scripts - These are custom scripts designed to interact with the DJI Osmo Action Pro
2. Robotdog UI - This is the UI core of this project, designed to interact with other aspects of the project.
3. Training - This is the Training component of the project, designed with the YOLO v26 model in mind.
4. Sample Yolo Scripts - A set of example scripts that conduct training utlizing CUDA. 

### Background

This project was introduced in tandem with the Company Month2Month as a potential solution. The purpose of this project is that it's designed to inspect and navigate (via remote) through an ideal office or home enviroment.
